## Project
The biggest project I worked on in Bootcamp was a Group Game. We were put into teams of 5 and tasked to make a game in two weeks. The parameters included that it had to be a one level maze game that was beatable in 5 minutes.

I was one of our group coders and our game designer. 

My contributions included:
1. Coding and designing booby traps
2. Coding and designing an Enemy to fight against
3. Designing our large game play map
4. Combining our group code into one functionning game

[Play Shadow Maze!](https://makecode.com/_55VFAF0m9hzd)
